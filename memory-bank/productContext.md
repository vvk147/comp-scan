# Product Context

## Why CompScan Exists
People use their computers daily without understanding patterns that hurt productivity, security, and wellbeing. Existing tools either send data to the cloud, cost money, or focus on only one aspect (monitoring OR security OR productivity).

CompScan is the unified, local, free, privacy-first solution.

## Problems It Solves
1. Invisible resource hogs draining battery and performance
2. Security misconfigurations that go unnoticed
3. Unproductive work patterns (excessive context switching, no breaks)
4. Disk space slowly filling up with caches and logs
5. Git repos with uncommitted work at risk of data loss
6. Late-night screen usage disrupting sleep

## User Experience Goals
- Single command to get started: `compscan scan`
- Progressive discovery: scan → observe → report → act
- Non-intrusive: observer runs in background at minimal cost
- Actionable: every insight either has a fix or a clear recommendation
- Safe: trust levels prevent accidental damage
- Beautiful: TUI dashboard with color-coded severity, progress bars
